What We Do Today

Marathon Intel is a community-powered tool that captures and analyzes network performance data from Marathon game sessions. Players run a lightweight capture agent that tracks ping, jitter, packet loss, and server connections in real time.

We're already collecting client-side telemetry across sessions — server IPs, RTT measurements, match duration, queue times, estimated tick rates, and regional server data. All of it feeds into dashboards and a public API the community can build on.

● Live capture agent (scapy / tshark)   ● Real-time dashboard   ● Session tracking   ● Region detection   ● REST API

The Vision

"I died here, and my ping was 140ms."

Imagine a 2D match replay — like PUBG.sh — but with a network performance layer on top. You scrub through a match, see your route on the map, and watch your ping, jitter, and packet loss in real time alongside combat events.

Players could finally see whether their network was hurting them in the moments that mattered. And Bungie gets crowdsourced signal on server quality across every region.

What We Need from the API

We're not asking for everything — just the data that lets us tie network quality to what's actually happening in-game. Post-match telemetry that complements the client-side capture we already have:

Player Positions Over Time

Enough to render a 2D match replay with player routes on the map.

Combat Events

Engagements, kills, damage — so players can see if their network was hurting them in the moments that mattered.

Match Metadata

Map, mode, duration, outcome — to replace the heuristics we're currently using to detect match boundaries from packet patterns.

Server Details

Region and tick rate — we're estimating tick rate from packet intervals right now, but authoritative data would let us build server quality dashboards the community can trust.

Why This Elevates the Game

Transparency builds trust. When players can see exactly what their connection looked like during a match, the conversation shifts from "these servers suck" to "here's the data." That's better for everyone — players understand their experience, and Bungie gets real signal on where to invest in infrastructure.

Destiny's community tools were a huge part of what made that ecosystem thrive. We'd love to help build the same for Marathon.
