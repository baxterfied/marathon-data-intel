Contribute Network Data

Run the Network Capture Agent while you play Marathon. It uses tshark to capture game traffic and auto-submits ping, jitter, and packet loss data to this dashboard.

Download netcapture.py — then run:

python netcapture.py --user-hash YOUR_NAME

Requires tshark installed. Run as admin. Use --dry-run to test without submitting.
